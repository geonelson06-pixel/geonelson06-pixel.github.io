1. Share one technical concept that you developed greater mastery over in this project. Demonstrate how you understand that concept by sharing your mental model of the concept. Then, show how you used that concept in your project.

My Response:

One technical concept I gained greater mastery over in this project was Flexbox layout. My mental model of Flexbox is that it lets me control how items are aligned and spaced along a main axis and a cross axis inside a container. Instead of positioning elements manually, I let the container decide how its children flow.

I used Flexbox throughout my project to organize sections like form rows and groups of inputs. By setting containers to display: flex and adjusting properties like flex-direction, gap, and alignment, I was able to create clean, responsive layouts without extra CSS complexity.
